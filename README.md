The Way To Flask
=======

###本文目标

通过讲解 Flask 以及它的一些常用扩展，介绍通用的用法以及需要注意的问题，帮助读者学会快速构建强壮的 Web(API) 服务器程序。

###声明

本文非原创，是改编自《[正则表达式30分钟入门教程](http://deerchao.net/tutorials/regex/regex.htm#mission)》，因为原作者的排版个人不是很喜欢，而且内容上个人觉得有些地方需要改进，所以在 gitbook 上开了一本书。
